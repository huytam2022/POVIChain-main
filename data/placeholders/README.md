# Data Placeholders

External measurement data is held outside the public source. Follow the schema contracts in `schemas/` and place artifacts as follows:

- `data/raw/` — per-device raw measurement JSON conforming to `schemas/raw_device_measurement.schema.json` (arrays of records). Filenames: `pi4_groth16.json`, `pi4_stark.json`, `esp32_verify.json`, and similar.
- `data/processed/device_profiles.processed.yaml` — processed calibration artifact conforming to `schemas/processed_calibration.schema.json`.
- `data/processed/device_profiles.index.json` — index pointing at artifacts, generated by the calibration pipeline.
- `data/processed/device_profiles.hash.txt` — SHA-256 hash of the processed artifact.
- `data/manifests/` — experiment manifests conforming to `schemas/experiment_manifest.schema.json`.

Replace placeholder series in `data/placeholders/processed/device_profiles.placeholder.yaml` with actual processed traces to enable regression-band reproduction. The placeholder file is **not** empirical calibration data and is clearly labeled as such.
